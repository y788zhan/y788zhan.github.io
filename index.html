University Program CS Program Search Algorithm

Description:

This program uses the language Scheme to achieve a search algorithm for data collected on the computer science programs from several universities in Ontario. The algorithm allows users to search for CS programs based on their own custom criteria, such as admission average, co-operative education availability, etc.

Addition Exercise Generator Using VBA

Description:

This program uses VBA to generate a user-customized number of addition questions, and includes a timer to record the time lapsed for the user to complete the questions. Results such as timer per question, and percentage correct are stored on another sheet.

Study of Relationship between Circular Cone Angle and Drag in Fluid Media

Description:

This essay explores the relationship between circular cones' half angle and their drag coefficient in viscous media. Cones of different half angles are made using plaster, and are dropped in water, sucrose solution, and isopropyl alcohol so study their free-fall motion.

Disruptive New Venture Concept: Time Block

Description:

Time Block is a time-management app that focuses on calendar-sharing and interactions between users, combining the concepts of online planners and social media. The application is designed for university students, as they have busy and variable schedules and high demand for convenient communication among their peers.
